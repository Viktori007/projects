# Игра "Блэк Джек" (21)

## Описание
Этот архив содержит программу на Python для игры в "Блэк Джек" (21). Игра реализована в текстовом интерфейсе с возможностью открытия в интегрированных средах разработки (IDE).

## Структура архива
- main.py: Основной файл программы, который запускает игру.
- Win.py: Файл с основной логикой игры.
- Button.py: Файл с классом
- Card.py: Файл с классом
- Desc.py:  Файл с классом
- Hand.py:Файл с классом

## Запуск программы
1. Откройте файл main.py в интегрированной среде разработки (IDE).
2. Запустите программу, нажав на кнопку "Запуск" или используя сочетание клавиш.
3. Следуйте инструкциям в окне игры в "Блэк Джек".

## Инструкции по игре
- В начале игры вам будет выдано две карты.
- Вы можете взять дополнительные карты или остановиться.
- Дилер будет автоматически продолжать брать карты до достижения определенной суммы.
- Цель игры - набрать сумму карт, как можно ближе к 21, не превышая ее.
- Если ваша сумма карт больше, чем у дилера и не превышает 21, вы выигрываете.

## Дополнительные замечания
- Убедитесь, что ваша IDE поддерживает запуск программы на Python.
- При возникновении проблем или вопросов обращайтесь к автору программы.
